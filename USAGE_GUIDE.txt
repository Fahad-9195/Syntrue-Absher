خطوة 1️⃣: الإعداد (تشغيل الخادم)
────────────────────────────────────────────────────────────────
1. اذهب إلى مجلد المشروع:
   C:\Users\abdal\Desktop\smart-security-absher

2. انقر مزدوج على: run.bat

3. سيظهر نافذة سوداء وتشاهد:
   ✅ [1] فحص البيئة الافتراضية...
   ✅ [2] تفعيل البيئة الافتراضية...
   ✅ [3] تثبيت المتطلبات...
   ✅ [4] تشغيل الخادم...
   
   📍 انتظر حتى ترى:
   INFO:     Uvicorn running on https://syntrue-absher.onrender.com

4. لا تغلق النافذة (اتركها مفتوحة طول ما تستخدم اللوحة)

خطوة 2️⃣: فتح لوحة التحكم
────────────────────────────────────────────────────────────────
افتح المتصفح (أي متصفح - Chrome, Firefox, Edge) والرابط:

   https://syntrue-absher.onrender.com/static/dashboard-simple.html

أو انقر مزدوج على الملف:
   افتح-لوحة-التحكم.url

═══════════════════════════════════════════════════════════════════

خطوة 3️⃣: استخدام اللوحة
────────────────────────────────────────────────────────────────

أ) شاهد الإحصائيات في الأعلى:
   • إجمالي الأحداث: عدد الأحداث الكلي
   • الأحداث الخطيرة: كم حدث خطير
   • التحذيرات: كم تحذير
   • آخر تحديث: الوقت

ب) اضغط على "▶️ تشغيل محاكاة":
   ✓ ستضاف 15 حدث عشوائي
   ✓ ستشاهد الخريطة تتحدث
   ✓ المخطط البياني يظهر البيانات
   ✓ الجدول يعرض الأحداث

ج) استخدم الفلاتر:
   • اختر مستوى الخطورة (خطر، تحذير، معلومة)
   • اختر الجهاز (door_sensor_1، motion_sensor_1، إلخ)

د) الأزرار الأخرى:
   • 🔄 تحديث يدوي: حدّث البيانات فوراً
   • ⏸ إيقاف التحديث: وقف التحديث التلقائي
   • 🗑️ مسح البيانات: احذف كل الأحداث

═══════════════════════════════════════════════════════════════════

خطوة 4️⃣: الروابط المفيدة الأخرى
────────────────────────────────────────────────────────────────

📚 توثيق API (لو تريد إضافة بيانات برمجياً):
   https://syntrue-absher.onrender.com/docs

   فيها جميع endpoints:
   • GET /api/events - احصل على جميع الأحداث
   • POST /api/events - أضف حدث جديد
   • POST /api/events/resolve_all - عيّن الأحداث كـ تم الحل
   • POST /api/events/reset - امسح جميع الأحداث

════════════════════════════════════════════════════════════════════

💡 نصائح مفيدة:
════════════════════════════════════════════════════════════════════

1. اترك نافذة الخادم مفتوحة دايماً (في الخلفية)
   - لو أغلقتها، اللوحة لن تشتغل
   - روح لنافذة الـ cmd وأغلقها، أو اضغط Ctrl+C

2. استخدم "تشغيل محاكاة" لإضافة أحداث اختبار

3. كل 3 ثواني التحديث يجري تلقائياً (لو ما ضغطت "إيقاف التحديث")

4. الخريطة توضح مواقع افتراضية في الرياض
